# WISATA DIENG

>Aplikasi Wisata Dieng merupakan aplikasi yang dibuat untuk masyarakat yang berisi sebaran tempat wisata dan deskripsi masing-masing tempat wisata tersebut.

## Komponen Pembangun Produk

Aplikasi ini dibuat menggunakan React Native Framework, AppsScript, dan Fontawesome5

### 1. React Native
React Native adalah sebuah framework open-source yang digunakan untuk mengembangkan aplikasi mobile secara multi-platform. Framework ini dikembangkan oleh Meta Platforms, Inc. dan didasarkan pada library JavaScript, React. React Native memungkinkan pengembang untuk menggunakan framework React bersama dengan kemampuan platform asli, sehingga memungkinkan pengembangan aplikasi untuk Android, iOS, macOS, Windows, dan platform lainnya.

### 2. AppsScript
Google Apps Script adalah platform yang dikembangkan oleh Google untuk mengotomatiskan, memperluas, dan terintegrasi dengan Google Workspace.

### 3. FontAwesome5
Font Awesome 5 adalah kit alat untuk situs web yang menyediakan ikon dan logo sosial.

### 4. Leaflet JS
Leaflet.js adalah sebuah pustaka JavaScript open source yang digunakan untuk membangun aplikasi pemetaan web.
